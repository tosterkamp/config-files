```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/tosterkamp/config-files/master/install.sh)"
```

#### via wget

```shell
sh -c "$(wget -O- https://raw.githubusercontent.com/tosterkamp/config-files/master/install.sh)"
```
